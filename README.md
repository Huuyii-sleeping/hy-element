# hy-element

一个基于 Vue 3 的轻量级 UI 组件库，包含常用组件和工具函数。


## ✨ 特性

- 基于 Vue 3 和 TypeScript 开发
- 支持按需引入和全量引入
- 提供常用组件：`Form`, `Input`, `Select`, `Message`, `Tooltip`, `Switch` 等
- 使用 Vite 构建，支持 UMD 和 ES 模块输出
- 内置国际化支持（`locale/`）
- 集成单元测试（Vitest）
- 支持主题定制（`theme/`）

## 🚀 安装

使用 npm：

```bash
npm install hy-element

```
使用pnpm：
```bash
pnpm add hy-elemenmt
```
